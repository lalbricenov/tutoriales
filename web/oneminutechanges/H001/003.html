    Creación de la base de datos en MongoDB Atlas con la colección de usuarios.

    Para crear la base de datos en MongoDB Atlas primero debemos ir a Atlas e iniciar sesión. Luego de eso debemos hacer click en browse collections.

    A continuación creamos una nueva base de datos, la cual vamos a llamar oneminute, y en ella una colección llamada usuarios.

    Creación de un index para que el correo sea único

    Dado que vamos a usar el correo para identificar de manera única a los usuarios, es importante que no existan dos usuarios con el mismo correo.

    Para esto se puede crear lo que se llama un index (índice) que lo que hace es permitir que la búsqueda por una columna en particular sea mucho más rápida. Con esa velocidad, mongodb puede verificar facilmente si el correo ya existe o no, y rechazar la adición del nuevo documento si el correo ya existe.

    Para crear el índice vamos a usar MongoDB Compass. Luego de conectarse al cluster, haga click en la base de datos oneminute, y posteriormente en la parte inferior donde dice mongosh.

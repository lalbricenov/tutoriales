    • Aplicación One Minute Changes usando Python, Mongodb y React

    En este tutorial vamos a realizar una aplicación movil usando Flask, mongodb Atlas, Heroku y React native.

    Aplicación

    Durante el aprendizaje de cualquier instrumento musical la práctica constante juega un rol muy importante. En particular para el aprendizaje de la guitarra es necesario estar en capacidad de cambiar entre un acorde y otro de manera ágil. One minute changes se refiere a una rutina de práctica que consiste en cambiar entre un par de acordes el mayor número de veces posible en un minuto.

    Requisitos

    Lea el articulo sobre los requisitos aquí.

    Personas

    • Pedro o María: hombre o mujer interesados en aprender a tocar guitarra que hacen uso de aplicaciones para mejorar su aprendizaje. Están interesados en practicar frecuentemente, de manera que puedan ver avances significativos.
    • Juan: profesor de guitarra que tiene varios estudiantes. Le interesa que sus estudiantes practiquen frecuentemente y ver su progreso, así como sus dificultades.

    Historias de usuario

    • H001 Como Pedro quiero iniciar sesión en la aplicación para que mi historial no se pierda incluso si cambio de dispositivo.
    • H002 Como Pedro quiero ver una tabla con el número de cambios en un minuto entre cada par de acordes para reconocer aquellos en los que necesito practicar más.
    • H003 Como Pedro quiero registrar una sesión de cambios de un minuto indicando la calidad y el número de cambios para poder verlos luego y entender el progreso que he tenido.
    • H004 Como Pedro quiero ver un gráfico como función del tiempo que muestre el avance en el número de cambios entre dos acordes para ver si estoy estancado.
    • H005 Como Juan quiero añadir a mis estudiantes a mi clase para poder ver su perfil, incluyendo su progreso en cada cambio de acordes.
    • H006 Como Juan quiero ver la lista de sesiones de práctica de cada estudiante organizadas por fecha para ver si ha sido responsable o no.

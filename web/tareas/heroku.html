    • Heroku

    Heroku es una plataforma en la cual se pueden poner en funcionamiento aplicaciones de diferentes tipos, en particular aplicaciones de python.

    En este tutorial vamos a publicar en Heroku una aplicación web creada usando flask que va a servir como el backend para otro tipo de aplicaciones, como por ejemplo una aplicación movil.

    Este tutorial está basado en una aplicación para llevar el registro de las tareas por hacer en diferentes materias.

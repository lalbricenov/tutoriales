    • Aplicación tareas usando Python y Mongodb

    En este tutorial vamos a realizar una aplicación web usando el módulo Flask de python, y el servicio mongodb Atlas. La funcionalidad principal de la aplicación será llevar un registro de las tareas por hacer, teniendo en cuenta la materia, la fecha y el detalle.

    Este tutorial está dividido en 3 partes:

    1. Creación de la aplicación base
    2. Configuración para la publicación en Heroku
    3. Inclusión de la base de datos
